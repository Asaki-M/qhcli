# qhcli

一个快速创建项目的 CLI 工具。

## 安装

```bash
npm install -g qhcli
# 或
yarn global add qhcli
# 或
pnpm add -g qhcli
```

## 使用

```bash
# 创建新项目
qhcli create <project-name>

# 示例
qhcli create my-project
```

## 功能特性

- 支持多种项目模板
- 自动规范化项目名称
- 自动初始化 Git 仓库
## 模板列表

- vite-plugin-easy: 一个简单的 Vite 插件模板
